# Archived status files

These are point-in-time status memos and reports from past work sessions (2025). They were moved out of the repository root on 2026-07-13 because they no longer reflect current state and were cluttering the root. They are kept here (rather than deleted) for historical reference.

Do not treat these as current. Live status now lives in:
- `README.md` — current project overview
- `CHANGELOG.md` — release history
- `public/docs/roadmap.md` — current roadmap
- `public/data/version.json` / `public/data/status.json` — current version data
